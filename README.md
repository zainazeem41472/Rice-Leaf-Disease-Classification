This project focuses on the **multi-class classification of rice leaf images** using deep learning. 
Given an input image, the system classifies it into one of several rice leaf disease categories or identifies it as a non-leaf image. 
The aim is to support early disease diagnosis by accurately categorizing rice leaf conditions using a custom CNN model.

The model performs **image classification** into the following classes:

- Healthy Leaf  
- Not Leaf (irrelevant images such as humans, chairs, pens, etc.)
